# Taller de Interacciones Web - Evaluador académico

## Propósito

Este proyecto amplía un evaluador académico desarrollado con HTML, JavaScript y Bootstrap. Permite registrar estudiantes, validar sus datos, calcular resultados individuales y presentar estadísticas generales del curso.

## Funcionalidades

- Registro de nombre, carrera y cuatro notas decimales.
- Validación de campos, rango de notas y nombres repetidos.
- Cálculo del promedio y del promedio especial sin la nota más baja.
- Clasificación del estado y del rendimiento académico.
- Cálculo de becas de acuerdo con la carrera y el promedio.
- Conteo de notas aprobadas, en recuperación y reprobadas.
- Recomendación académica personalizada.
- Historial, búsqueda y ranking de estudiantes.
- Estadísticas generales, mejor promedio y menor promedio del curso.
- Presentación del objeto del estudiante en formato JSON.

## Tecnologías utilizadas

- HTML5
- JavaScript
- Bootstrap 5.3.3 mediante CDN

## Ejecución

1. Descargue o clone el repositorio.
2. Abra la carpeta del proyecto.
3. Abra `index.html` en un navegador web.
4. Complete todos los campos y pulse **Evaluar**.

No se requiere instalación ni servidor. Para visualizar los estilos, se necesita conexión a Internet porque Bootstrap se carga desde CDN.
